# Products Catalog App

A responsive product catalog built with React and TypeScript.

## 🛠️ Tech Stack

- React
- TypeScript
- CSS (Grid & Flexbox)
- Vite / Create React App (if applicable)

## How to run

1. git clone https://github.com/merscarlett/aqvex-shop
2. cd aqvex-shop
3. npm install
4. npm run dev
